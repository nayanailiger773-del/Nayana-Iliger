Machine Learning is a way of teaching computers to learn patterns from data instead of giving them fixed step by step instructions. In traditional programming, we write rules and give input data to get an output. But in Machine Learning, we provide input data along with the correct answers, and the system automatically discovers the rules or patterns that connect them. Once it learns these patterns, it can use them to make predictions or decisions on new data. In simple terms, instead of programming every rule manually, we let the computer learn the rules from examples.



*Supervised Learning
Supervised Learning is when the data already has correct answers (labels). The model learns from input plus output and then predicts for new data.

Example :
Study hours → Marks
We already know the marks. The model learns the pattern.

*Unsupervised Learning
Unsupervised Learning is when the data has no labels. The model tries to find hidden patterns or groups by itself.
The model discovers structure.
Example :
Customer purchase data → The model groups similar customers together.

*Reinforcement Learning
Reinforcement Learning is when an agent learns by interacting with an environment using rewards and penalties.
The model learns by trial and error.
Example:
A game-playing AI gets:
+10 for winning
−5 for losing
It learns the best strategy over time.
